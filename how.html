How this Dashboard Was Built

This Corona virus dashboard was built using the DreamFactory iPaaS solution and Tabulator JavaScript library. DreamFactory streamlines the time and effort required to integrate data stored within sources such as databases, file systems, IoT devices, and even mainframe computers by generating well-defined, well-documented, secure, and scalable APIs.

The DreamFactory platform is used by thousands of organizations around the globe, and is particularly popular in industries such as banking, logistics, manufacturing, and software services. Companies typically run the platform in their cloud (AWS, Azure, etc.) or within their own data center, however many take advantage of the hosted offering.

This particular DreamFactory-backed dashboard was built in approximately 90 minutes, including domain name registration, DNS configuration, and website configuration (on Digital Ocean).
